Overview

Q3N is a lightweight, machine-readable format for storing quotations with their source information in a standardized way. It uses triple-slash delimiters and URI-formatted source references to create a simple yet powerful system for citation management.

Basic Syntax

/// https://example.com/article /// cite/article:
The quoted text goes here.
Multiple paragraphs are fine.
\\\

URI-based Sources

Use standard URI schemes like https:// or custom schemes like isbn://, doi://, pubmed://, orcid://, and more.

Tag Hierarchy

Organise entries with slash-delimited tags: cite/article, note/idea, cite/book.

Simple Parsing

The format is designed to be easily parsed line-by-line in any programming language. Implementations exist in Python and JavaScript.

Examples

Web Citation

/// https://example.com/article /// cite/article:
The quoted text goes here.
\\\

Book Quote

/// isbn://9780547249643;'1984';'George Orwell';1949
War is peace. Freedom is slavery. Ignorance is strength.
\\\

Person Reference

/// q3n://grace:@hopper;grace@navy.mil;'Grace Hopper'
The most dangerous phrase in the language is, "We've always done it this way."
\\\

Academic Paper

/// doi://10.1126/science.187.4176.433
The more the universe seems comprehensible, the more it also seems pointless.
\\\

Getting Started

Install

pip install pyside6
git clone https://github.com/thuruht/q3n
cd q3n
pip install -e .

Create your first collection

q3n init my-quotes.q3n
q3n create my-quotes.q3n
q3n show my-quotes.q3n
